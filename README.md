# VHDL Components

The content of this repository is free to use  

Components include:

- Multiplexer
- Demultiplexer
- Adder
  - Full-Adder
  - Half-Adder
- Information Storage (clk rising edge, no preset, no async clear)
  - JK-FF
  - D-FF
  - T-FF
- Counter
  - 4Bit (uses JK-FF + async Reset)
    - normal
    - arithmetic
  - 3 Bit prime number counter
    - case based
